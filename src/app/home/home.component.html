PizzaBox

PizzaBox

PizzaBox was my solo project at Ironhack. After two weeks learning how to use the Express.js framework in combination with Node.js and MongoDB, I created an app to help pizza people everywhere sell off a pizza they need to sell off! Users can register as a customer or as a restaurant. Restaurants can create and modify their PizzaBoxes as well as set a price. Customers can purchase PizzaBoxes using the Stripe API.


Deadline: 1 week

Technologies used: JavaScript (Handlebars.js, Node.js, Express.js), MongoDB, Stripe API, HTML5, SCSS, Bootstrap 4 Material Kit

Tournamaker

Tournamaker

Tournamaker was our final project at Ironhack. With the rise of esports and the longstanding love of tournaments, Tournamaker is intended to make it easy for people to form teams and join a tournament. My team integrated over 17 components using Angular 6 in order to create, read, update, and delete information from the server, as well as pass information in between them as necessary. Tournamaker uses the full MEAN Stack, and the database has multiple relationships involved, with three different models forming relationships with each other.


Deadline: 1.5 weeks

JavaScript (Node.js, Express.js, Angular 2), MongoDB, HTML5, CSS
